#**Getting Started**
---
Proyek ini dibangun di atas OS Windows 11 menggunakan IDE Visual Studio Code. Rincian mengenai *library* yang digunakan untuk proyek ini dapat dilihat melalui *file* requirements.txt. 

Untuk menjalankan proyek ini di PC lokal Anda, **pastikan untuk memperhatikan beberapa ketentuan berikut**:
1. Unduh requirements.txt dengan menjalankan kode berikut di terminal: pip install -r requirements.txt
2. Untuk menjalankan aplikasi, jalankan kode berikut di terminal: streamlit run st_page1.py
3. **Jangan** ubah direktori atau struktur aplikasi untuk menjamin aplikasi berjalan dengan baik (bila Anda ingin menggunakan proyek ini sebagai *template*, silahkan ubah direktori sesuka Anda).
4. Apabila Anda **menjalankan notebook NLP.ipnyb**, pastikan untuk **menjalankan repickle_custom_functions.py** sebelum menjalankan streamlit. Ini dilakukan agar fungsi yang dibutuhkan dapat berjalan di module main direktori Anda.
